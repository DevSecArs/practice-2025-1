О проекте

Данный проект представляет собой веб-приложение для управления пользовательскими кошельками и просмотра истории транзакций. Оно разработано с использованием Python и фреймворка Flask. Основная цель проекта — предоставить удобный интерфейс для взаимодействия пользователей с их цифровыми активами.

Основной функционал

  • Регистрация новых пользователей.
  • Авторизация с использованием логина и пароля.
  • Просмотр текущего баланса кошелька.
  • Отображение истории транзакций с подробностями (отправитель, получатель, сумма).
  • Займ криптовалюты (заглушка для тестирования функционала).
  • Запросы на осуществление транзакций.
  • Автоматическое обновление данных в реальном времени (каждые 10 секунд).
  • Поддержка светлой и тёмной темы для удобства пользователей.

Технологии

Для реализации проекта использованы следующие технологии:

  • Flask — фреймворк для построения веб-приложений на Python.
  • Bootstrap — CSS-фреймворк для стилизации и создания адаптивного интерфейса.
  • AJAX — для динамического обновления данных без перезагрузки страницы.
  • LocalStorage — для сохранения пользовательских настроек (тема интерфейса).

Как это работает?

Пользователь регистрируется или входит в систему, после чего ему доступен интерфейс для просмотра текущего состояния кошелька и истории транзакций. Данные о балансе и транзакциях запрашиваются с сервера в режиме реального времени и отображаются в удобном формате.
